# Olist E-commerce Analytics: Delivery and Customer Satisfaction

## Project idea
Order-level analytics for Olist marketplace: delivery performance, delays, and how they affect customer reviews and key business metrics.

## Dataset
Olist Brazilian E-Commerce public dataset (CSV tables).

Main tables:
- olist_orders_dataset: order lifecycle timestamps + status
- olist_order_items_dataset: items, price, freight, seller_id, product_id
- olist_order_payments_dataset: payment type and value
- olist_order_reviews_dataset: review_score
- olist_customers_dataset: customer location (state, zip prefix)
